Neutral environment assets for the GLB bath configurator.

neutral-reflection-studio.svg is designed to improve gloss readability with neutral white/grey reflection panels.

Avoid warm, green, brown or strongly coloured HDR environments because they distort colour preview.

If the current model-viewer/browser combination does not accept SVG as environment-image, export this SVG to a 2:1 PNG/JPG and use that file instead.
